# @mmmbuto/codex-cli-termux

Repackaged builds of the upstream Codex CLI with minimal compatibility patches
for Android Termux.

## Version

- **Package**: `@mmmbuto/codex-cli-termux`
- **Version**: `0.98.0-termux`
- **Supported**: Android Termux (ARM64) only

## Install

```bash
npm install -g @mmmbuto/codex-cli-termux
```

## Verify

```bash
codex --version
codex exec --help
```
